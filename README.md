# PICYBOO NFT Meta Normalizer

Normalizes NFT metadata (ERC-721/1155 style) to a compact schema.

**Status:** Archived • Demo only • No support  
**Trademark:** PICYBOO — US 97338199 / CA 2173851

## Install
```bash
npm i
npm link   # optional to expose 'pbnft' globally
```

## CLI
```bash
pbnft normalize ./examples/sample.json
```

## Output schema
```json
{"name":"","description":"","image":"","attributes":[],"hash":""}
```
